# InvesTechy Frontend

Dokumentasi utama frontend ada di file berikut:

- [FRONTEND_DOCUMENTATION.md](./FRONTEND_DOCUMENTATION.md)

Isi dokumentasi tersebut mencakup:

- arsitektur frontend
- struktur folder `src`
- routing halaman
- implementasi React Redux dan Redux Thunk
- daftar semua lokasi `useState` dan `useEffect`
- ringkasan service API, context, provider, dan local storage

Perintah menjalankan project:

```bash
npm install
npm run dev
```

Perintah build:

```bash
npm run build
npm run preview
```
